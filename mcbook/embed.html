How to Install
Works on any website — WordPress, Squarespace, Wix, or custom HTML.
1
Copy the embed code above using the Copy button.
2
Open the page on your website where you want the booking widget to appear.
3
Paste the code into the HTML of that page — anywhere inside <body>. On WordPress, use a Custom HTML block. On Squarespace, use a Code Block.
4
The widget appears automatically and adapts to your site's colour scheme — no extra styling needed.
